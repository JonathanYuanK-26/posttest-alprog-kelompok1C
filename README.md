## 📁 Deskripsi Grafik

File `PostTest_K1_KelasC_Grafik.ipynb` berisi gabungan analisis dari Grafik 1 sampai Grafik 5 dalam satu notebook, dibagi menjadi empat kategori utama.

---

### Kategori A — Agregasi

Menghitung rata-rata harga rumah (`price`) berdasarkan `furnishingstatus`. Hasil ditampilkan dengan **Bar Chart** sehingga perbedaan rata-rata harga antara rumah *furnished*, *semi-furnished*, dan *unfurnished* dapat dibandingkan dengan mudah.

---

### Kategori B — Tren/Filter

Memfilter data rumah yang berada di jalan utama (`mainroad = "yes"`) dan memiliki AC (`airconditioning = "yes"`). Setelah difilter, program menghitung rata-rata harga rumah dari kelompok tersebut dan menampilkannya dalam **Bar Chart**.

---

### Kategori C — Korelasi

Menganalisis hubungan antara luas tanah (`area`) dan harga jual rumah (`price`). Nilai korelasi dihitung untuk melihat kekuatan hubungan kedua variabel, lalu **Scatter Plot** digunakan untuk memperlihatkan pola persebaran data.

---

### Kategori D — Distribusi

Mengidentifikasi pencilan (*outlier*) pada data harga rumah (`price`) menggunakan metode **IQR**. **Boxplot** digunakan untuk menampilkan distribusi harga rumah sekaligus menunjukkan titik data yang termasuk pencilan.
